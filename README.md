# Audio Utility App (Personal Project)

This app was created for **two main reasons**:

## 1. Solving a Problem

Music platforms are good, but they all have limitations:

### Spotify
- Requires payment to remove ads
- Doesn’t include many *unofficial releases* (edits, remixes, bootlegs, fan uploads, etc.)

### YouTube
- Requires payment to remove ads
- No convenient built-in way to:
  - download audio-only
  - shuffle downloaded tracks like a music library

### Both Platforms
- No real-time audio modification tools such as:
  - pitch shifting
  - bass boost
  - reverb
  - other audio effects

This app exists because I wanted a single place that supports all of the above.

# Main Findings

### AI works best with small code sections
AI is usually very effective when dealing with smaller snippets of code.  
As the codebase grows, output quality tends to degrade.

### Prompting matters more than expected
Creativity often depends on phrasing. For example:

❌ *"Fix this problem creatively."*  
✅ *"Generate your own approach to (x) with (y) parameters."*

The second style consistently produces better results.

### AI is great at implementing guided ideas
If you already have a rough idea of how something should work and suggest it, AI will implement it correctly about 9/10 times.
